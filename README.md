# 我的餐廳清單

## 專案功能
* 使用者可以在首頁看到餐廳簡易訊息
* 使用者可以點及餐廳圖片查看餐廳詳細訊息
* 使用者可以再詳細訊息頁面連結到googlemap
* 使用者可以在首頁用餐廳類別搜尋餐廳
* 使用者可以在首頁用餐廳名稱搜尋餐廳

## 開始使用
1.確認已安裝Node.js  
2.將本專案clone進本地端
3.安裝套件
```js
npm i express@4.17.1 express-handlebars@3.0.0 mongoose@6.0.4 method-override@3.0.0
```
4.執行程式
```js
npm run start
```
5.當cmd出現以下這行表示啟動成功
```js
Express is listening on localhost:3000
```
6.接著在瀏覽器輸入
```js
http://localhost:3000
```
